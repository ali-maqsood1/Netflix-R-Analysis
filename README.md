# 📊 Netflix Data Analysis using R

## Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using R.

The objective is to clean the dataset, analyze trends, and create visualizations that provide insights into Netflix's content library.

---

## Dataset

Dataset:
Netflix Movies and TV Shows

Contains information about:

- Movies
- TV Shows
- Directors
- Cast
- Countries
- Ratings
- Genres
- Duration
- Release Year

## Key Insights

- Movies make up the majority of Netflix content.
- The United States contributes the largest number of titles.
- Netflix experienced rapid content growth after 2015.
- Drama and International Movies are among the most common genres.
- TV-MA is the most frequent content rating.
- Most movies are between 80–120 minutes long.
- Some metadata fields such as director and country contain missing values.
